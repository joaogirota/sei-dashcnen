Política de Privacidade — SEI+DashCNEN

Última atualização: 04 de maio de 2026

1. O que é esta extensão

SEI+DashCNEN é uma extensão para o navegador Chrome que adiciona um painel de acompanhamento de prazos, estatísticas e equipe ao Sistema Eletrônico de Informações (SEI) utilizado pela Comissão Nacional de Energia Nuclear (CNEN).

2. Dados coletados e armazenados localmente

A extensão armazena os seguintes dados exclusivamente no seu navegador (via chrome.storage.local), sem transmiti-los a servidores externos:

  • URL base do sistema SEI da sua instituição
  • Preferências de configuração do dashboard (período, filtros, tema)
  • Cache de dados de processos e estatísticas para exibição no painel
  • Estado das notificações e contagem de processos não visualizados

3. Acesso a páginas do SEI

A extensão injeta scripts nas páginas do SEI para ler informações de processos, prazos e equipe diretamente da interface do sistema. Esses dados são utilizados apenas para exibição no painel da extensão e não são enviados a terceiros.

4. Funcionalidade de análise por IA (opcional)

Se a funcionalidade de análise por inteligência artificial estiver configurada e habilitada pelo usuário, a extensão poderá enviar conteúdo de documentos de processos SEI para um servidor de análise externo (configurado pelo próprio usuário ou pela instituição). Essa funcionalidade:

  • É desativada por padrão e requer configuração explícita
  • Envia apenas o conteúdo dos documentos selecionados para análise
  • O endereço do servidor é definido pelo próprio usuário/instituição

O usuário é responsável por verificar a política de privacidade do servidor de análise configurado.

5. Dados não coletados

Esta extensão não coleta, transmite nem compartilha:

  • Dados pessoais de identificação (nome, CPF, e-mail)
  • Credenciais de acesso (usuário ou senha do SEI)
  • Histórico de navegação
  • Dados de telemetria ou uso da extensão

6. Permissões utilizadas

  • storage: Armazenar preferências e cache localmente no navegador
  • notifications: Exibir alertas de novos processos na unidade
  • alarms: Agendar verificações periódicas em segundo plano
  • tabs: Abrir o painel e criar abas temporárias para extração de dados do SEI
  • scripting: Reinjetar scripts de comunicação após atualização da extensão
  • Host *.br: Operar em qualquer instalação do SEI em domínios .br, pois cada órgão utiliza seu próprio subdomínio

7. Contato

Dúvidas sobre esta política podem ser enviadas para: joaogirota@gmail.com

Privacy Policy — SEI+DashCNEN

Last updated: May 4, 2026

1. What this extension does

SEI+DashCNEN is a Chrome browser extension that adds a dashboard for tracking deadlines, statistics, and team information to the Electronic Information System (SEI) used by the Brazilian National Nuclear Energy Commission (CNEN).

2. Data collected and stored locally

The extension stores the following data exclusively in your browser (via chrome.storage.local) and does not transmit it to external servers:

  • Base URL of your institution's SEI system
  • Dashboard configuration preferences (period, filters, theme)
  • Cached process and statistics data for display in the panel
  • Notification state and unread process count

3. Access to SEI pages

The extension injects scripts into SEI pages to read process, deadline, and team information directly from the system interface. This data is used solely to display information in the extension panel and is never sent to third parties.

4. AI analysis feature (optional)

If the AI analysis feature is configured and enabled by the user, the extension may send the content of SEI process documents to an external analysis server (configured by the user or institution). This feature:

  • Is disabled by default and requires explicit configuration
  • Sends only the content of documents selected for analysis
  • The server address is defined by the user/institution

The user is responsible for reviewing the privacy policy of the configured analysis server.

5. Data not collected

This extension does not collect, transmit, or share:

  • Personal identification data (name, national ID, email)
  • Access credentials (SEI username or password)
  • Browsing history
  • Telemetry or extension usage data

6. Permissions used

  • storage: Store preferences and cache locally in the browser
  • notifications: Display alerts for new processes in the unit
  • alarms: Schedule periodic background checks
  • tabs: Open the panel and create temporary background tabs to fetch SEI data
  • scripting: Re-inject communication scripts after extension updates
  • Host *.br: Operate on any SEI installation under .br domains, as each agency uses its own subdomain

7. Contact

Questions about this policy may be sent to: joaogirota@gmail.com
